Clear Linux* OS - An Introduction and Beyond

02 May, 2019

Please RSVP to attend the first Clear Linux* OS Meetup

Where

https://goo.gl/maps/9qKH7vReehFBo9Yk7

Intel Hawthorn Farm - HF3 Auditorium
5200 NE Elam Young Pkwy, Hillsboro, OR 97124

When

Wednesday, May 22, 2019
3:00 PM to 8:30 PM

Details

Meetup: https://www.meetup.com/Open-Source-Technology-Center-PDX/events/260768637/

We invite you to join us for the first in a series of Intel Clear Linux OS MeetUps. The aim of this initial MeetUp is to introduce you to the Clear Linux* project and help you learn how to better use the Clear Linux OS in your everyday job. Light refreshments and dinner provided.

What to Expect

You will:
1. Leave with an understanding of the value of the Clear Linux OS to the industry (performance, security, customization, and manageability)
2. Learn how to use the Clear Linux OS as your main workstation and deliver software
3. Receive a non-technical introduction to features and their benefits

Proposed agenda (subject to change)

3:00 Check in, refreshments, networking
3:30 Clear Linux Project Overview
3:45 Clear Linux OS Architecture
4:45 Demo: Deep Learning Reference Stack using Clear Linux OS
5:15 Demo: Swupd
5:45 Dinner Break
6:15 Hands on workshop
8:15 Panel
