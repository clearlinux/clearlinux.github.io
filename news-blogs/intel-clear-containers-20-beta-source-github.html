Intel® Clear Containers 2.0 beta source on GitHub

08 Aug, 2016

The source code for the Clear Containers OCI runtime and matching QEMU are now available on GitHub:

  • https://github.com/01org/cc-oci-runtime
  • https://github.com/01org/qemu-lite

Work is actively on-going, being driven through the GitHub issues and milestone methods, and a dashboard overview of the project can be seen on the waffle website.

Project quality is being maintained through a pull-request/review workflow alongside utilizing the Travis CI Continuous Integration system:

Instructions are available in the GitHub repository for installing Clear Containers into Docker* 1.12 running under Clear Linux* project. Documentation for manual installation under other Linux distributions will be uploaded soon whilst we work on the generation and publishing of the installation packages for many of the popular Linux distributions.

Taking part - an open project

With the release of the code on GitHub as a fully open GPL v2.0+ licensed project, we are welcoming input and involvement from the community to further push the boundaries of running container projects in Virtual Machines in a fast, efficient and secure manner.

Whether you are a coder, tester, user or documenter, feel free to join the Clear Containers community on GitHub and contribute. Details, including IRC channel and mailing list, can be found in the CONTRIBUTING document in the source:

Next steps

The next steps in the project are to complete some of the outstanding GitHub issues and move us out of the beta phase. A 'Container 2.0' milestone/sprint can be found on GitHub, and tracked on the waffle pages, to monitor progress. No doubt a new blog will follow here when that milestone is reached!
